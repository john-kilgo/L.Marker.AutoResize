L.Marker.AutoResize

A Leaflet plugin for automatic marker resizing based on zoom level

Get the plugin

The most up to date code is found in marker-resize.js located at the github repo. Currently tested with Leaflet v0.7.7 & v1.0.0-rc.1.

Example

View the example on a separate page here.

Include the plugin in the head section of your page


<script src="/path/to/marker-resize.js"></script>

Create icons


var MyIcon = L.Icon.extend({
	options: {
		iconUrl: 'path/to/image.png'
	}
});

var icon1 = new MyIcon({iconSize: [30, 30], iconAnchor: [15, 30]}),
	icon2 = new MyIcon({iconSize: [50, 50], iconAnchor: [25, 50]}),
	icon3 = new MyIcon({iconSize: [60, 60], iconAnchor: [30, 60]});

Create a marker

Required option is iconArray, an array of exactly three icons.


L.autoResizeMarker([42.5, -71.239], {iconArray: [icon1, icon2, icon3]}).addTo(mymap);
